# RescueNet AI

**AI-Powered Disaster Response Platform for Viasat Space For Good India 2026**
**Category: Multi-Orbit Constellations**

---

## 🛰️ Overview

RescueNet AI is a production-quality web application demonstrating how AI and Multi-Orbit Satellite Constellations (LEO, MEO, GEO) can save lives during natural disasters by maintaining communication, coordinating rescue teams, and providing intelligent decision support.

## 🚀 Features

- **AI Command Center Dashboard** — Mission intelligence with real-time stats
- **Interactive Disaster Map** — Leaflet-powered with animated heatmaps
- **AI Satellite Recommendation Engine** — Selects optimal orbit based on 8 factors
- **Live Multi-Orbit Network Visualization** — Animated LEO/MEO/GEO constellation
- **Automatic Satellite Switching** — AI-managed orbit handoff with logs
- **AI Disaster Prediction** — 5 disaster types with risk scoring
- **AI Rescue Prioritization** — ML ranking by medical urgency
- **AI Resource Allocation** — Optimal team/equipment deployment
- **Drone Fleet Management** — 5 UAVs with thermal/night vision
- **IoT Sensor Monitoring** — 8+ sensor types with live feed
- **Weather Intelligence** — 5-day forecast + AI impact analysis
- **Emergency SOS System** — Satellite-relayed with offline queuing
- **Analytics Dashboard** — Chart.js visualizations
- **RescueNet AI Assistant** — Chatbot for mission queries
- **Offline Communication** — Store & sync messaging

## 🛠️ Tech Stack

- **Frontend**: React 19, Tailwind CSS, Vite
- **Charts**: Chart.js + react-chartjs-2
- **Map**: Leaflet + react-leaflet
- **Icons**: Lucide React
- **Animation**: Framer Motion

## 📦 Setup

```bash
npm install
npm run dev       # Development server
npm run build     # Production build
npm run preview   # Preview production build
```

## 🌐 Deployment

Deploy to Vercel:
```bash
vercel --prod
```

The `vercel.json` file handles SPA routing.
